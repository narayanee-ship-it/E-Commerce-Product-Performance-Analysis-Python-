# E-Commerce-Product-Performance-Analysis-Python-

📖 Overview

Analyzed e-commerce transaction data using Python to evaluate product performance across categories and uncover business insights.

🎯 Objective

To understand how different product categories contribute to sales and revenue, and identify the right metrics to measure their performance.

🛠️ Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn

🧾 Dataset
239 transactions
6 product categories
Includes quantity and revenue data

🔍 Analysis Performed
Category-wise sales analysis
Revenue comparison across categories
Data visualization using charts
Performance evaluation by category

💡 Key Insight
Clothing had the highest units sold
Electronics had the lowest volume but generated 4× higher revenue

👉 This shows that:
High sales volume does not always mean high revenue
Different product categories require different KPIs
